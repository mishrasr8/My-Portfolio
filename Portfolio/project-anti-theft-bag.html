🥉 2nd Runner-Up – Phase Shift 2025

Anti-Theft Bag for Public Transportation

An embedded security system that detects unauthorized access and triggers real-time alerts to protect commuters in public transport.

Embedded Systems Sensors Circuit Design Problem Solving

Project Overview

The Anti-Theft Bag is an embedded hardware solution designed to tackle the growing problem of theft in public transportation systems. By integrating sensors directly into a bag, the system detects any unauthorized access attempts and immediately triggers alert mechanisms, giving the owner real-time notification.

This project was built and presented at Phase Shift 2025, where it was awarded 2nd Runner-Up among competing engineering teams — validating its real-world impact and technical execution.

Problem Statement

Theft in crowded public transport (buses, metros, trains) is a widespread issue that often goes undetected due to the busy environment. Traditional bags offer no security feedback to the owner when tampered with.

Our Solution

An embedded system inside the bag that monitors zipper/flap movement via sensors, triggering an audible alarm and/or notification whenever unauthorized access is detected — instantly alerting the owner.

Key Features

  • Real-time sensor detection of bag opening attempts
  • Audible alarm circuit triggered on unauthorized access
  • Low power consumption — suitable for portable use
  • Compact and discreet design — embedded within the bag structure
  • Rapid prototyping approach with iterative sensor tuning
  • Designed for robustness in crowded, high-vibration environments

Technical Implementation

Embedded System Architecture

The system uses microcontroller-based logic to continuously poll sensor states. Upon detection of a threshold breach (bag being opened), the firmware triggers the alarm circuit within milliseconds.

Sensor Integration

Carefully selected sensors were integrated at critical access points of the bag. Threshold tuning was done empirically to minimize false positives while ensuring high sensitivity to genuine access attempts.

Alarm Circuit

The alert mechanism uses a dedicated alarm circuit designed to produce a loud, attention-grabbing alert. The circuit was tested across different power conditions to ensure consistent performance.
